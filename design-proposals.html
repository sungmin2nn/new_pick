🎯 인터랙션 가이드

실제로 클릭해볼 수 있는 인터랙티브 프로토타입입니다. 아래 순서대로 테스트해보세요.

① Depth 1 — 전략 전체 합
"내일 후보" 섹션 각 전략 헤더에 평균점수 + 평균수익률 + 종목 개수 표시.
Alpha는 초기에 펼쳐져 있음 (1위).
② Depth 2 — 전략별 종목
Beta/Gamma/Delta/Echo 헤더 클릭 → accordion 펼치기/접기.
각 종목: 순위 + 이름 + 코드 + 점수 + 수익률.
③ Depth 3 — 종목 상세
종목 row 클릭 → 바텀시트 drawer 열림.
drawer에: 현재가, 점수 분해(프로그레스 바), 5일 수익률, 액션 버튼.
닫기: ✕ 버튼, 배경 터치, 또는 아래로 스와이프.

✨ 특징

컨텍스트 유지: 페이지 이동 없이 모든 정보 접근
스크롤 최소화: 관심 전략만 펼쳐서 필요한 정보만 scan
모바일 네이티브: iOS 바텀시트 스타일 drawer
한국 금융 컨벤션: 상승 빨강 / 하락 파랑
dashboard-skill 준수: JetBrains Mono 숫자, gap-4, 작은 라벨

🔧 적용 후 구조

이 디자인을 실제 index.html에 적용하면:

Arena 탭 = 이 화면 (5 sub-tabs 제거)
BNF 탭 = 기존 iframe (bnf_dashboard.html)
→ 메인 2탭, Arena는 단일 스크롤 + accordion + drawer
