🎯 인터랙션 가이드

모든 뎁스가 아코디언입니다. 페이지 이동이나 모달 없이 인플레이스로 정보가 펼쳐집니다.

① Depth 1 — 전략 전체 합
각 전략 헤더에 평균점수 · 평균수익률 · 종목 개수.
Alpha는 초기에 펼쳐져 있음 (1위).
② Depth 2 — 전략별 종목
전략 헤더 ▶ 클릭 → 종목 리스트 펼침.
종목별: 순위 + 이름 + 코드 + 점수 + 수익률 + ›.
③ Depth 3 — 종목 상세 (중첩 Accordion)
종목 row › 클릭 → 같은 위치에서 상세 패널 펼침.
현재가, 점수 분해, 5일 수익률, 액션 버튼.
다른 종목 클릭 시 이전 종목은 자동으로 닫힘 (한 전략 내).

✨ Drawer 대비 장점

풀스크린 가림 없음: 다른 종목과 비교하면서 보기 쉬움
스크롤 위치 보존: 닫고 다른 종목 열어도 위치 유지
중첩 컨텍스트 명확: 어느 전략의 종목인지 시각적으로 명확
모바일 친화: 한 손 엄지로만 조작 가능
