# Strategy
Trading Strategy
단타 매매 전략 이름. 카테고리(돌파/역추세/수급 등)와 참신성 ★, 승급 상태가 함께 표시됩니다.
 		Return
Total Return Percentage
총 수익률. 백테스트 기간 동안 돈이 얼마나 늘었는지.
1,000만원 → 1,352만원 = +35.2%
 		Sharpe
Sharpe Ratio
샤프 비율 — 위험 대비 수익이 얼마나 좋은지. 클수록 좋음.
<1 별로 / 1~2 괜찮음 / 2~3 좋음 / >3 매우 우수. 짧은 기간엔 과대평가 주의
 		Sortino
Sortino Ratio
소르티노 비율 — Sharpe와 비슷하지만 "아래로 떨어지는 위험"만 고려. 상승 변동은 좋은 것으로 봐서 더 현실적.
Sharpe보다 숫자가 보통 더 큼. 기준: >1 괜찮음
 		MDD
Max Drawdown
최대 낙폭 — 가장 높았을 때부터 가장 낮았을 때까지 얼마나 떨어졌는지. 0에 가까울수록 좋음.
1,000만원 → 1,200만원 → 1,050만원이면 MDD = -12.5%
 		Win Rate
Win Rate
승률 — 100번 매매 중 몇 번 이겼는가. 50% 이상이면 괜찮음.
승률이 높아도 한 번 크게 지면 손실일 수 있음 → Profit Factor와 함께 봐야 함
 		PF
Profit Factor
이익 요인 — 총 번 돈 / 총 잃은 돈. 1보다 크면 흑자.
100만원 벌고 50만원 잃으면 PF = 2.0. 기준: >1.5 괜찮음 / >2 좋음
 		Trades
Number of Trades
매매 횟수 — 백테스트 기간 동안 총 몇 번 사고 팔았는지. 거래가 적으면 통계 신뢰도 낮음.
최소 10번 이상이어야 통계적으로 의미 있음
Phase 8 · Realistic Simulation

3-Tier 현실 검증 · Nominal / Intraday / Probabilistic

기존 일봉 시뮬(Nominal)은 장중 경로를 모르기 때문에 실제보다 낙관적입니다. Phase 8에서는 (1) 1분봉 실측(Tier 1, 6일 한계), (2) OHLC 기반 확률적 exit(Tier 2, 장기), (3) 분봉 ↔ 확률 Calibration, (4) Bootstrap p-value · Walk-Forward · KOSPI alpha 통계 검증을 더해 "얼마나 진짜인가"를 측정합니다.

# Strategy
Trading Strategy
전략 이름. walk-forward 수익률 기준 정렬.
 		Nominal
Nominal Return (Day-bar)
기존 일봉 시뮬 결과. 장중 경로를 모르기 때문에 고점 도달 여부를 가정해서 계산 — 보통 가장 낙관적이거나 보수적인 방향으로 편향됨.
 		Tier 1
Tier 1 · Intraday Actual
1분봉 실측 시뮬. 09:00 예약 매수, 분봉 순차 탐색으로 +5% 익절 / -3% 손절 / 종가 청산 판정. 거래비용(수수료 0.015%×2 + 거래세 0.18%) 반영. 가장 현실에 가까움 (Ground Truth).
Yahoo Finance 1m API — 최근 6거래일만 가능
 		Tier 2
Tier 2 · Probabilistic Day-bar
일봉 OHLC로부터 "고점이 먼저였을 확률"을 추정해 확률 가중 기대값 계산. 장기간(1개월/1년)에 쓸 수 있는 대신 실측이 아닌 추정.
p_high_first = 0.5 + 0.8 × (추세 / 변동폭)
 		Calibrated
Calibrated Return
Tier 2 × (Tier 1 / Tier 2) 보정 계수. 분봉 실측 수준에 맞춘 장기 추정치.
 		WF
Walk-Forward Return
롤링 학습/테스트 윈도우로 과적합 체크. 전체 평균과 WF 평균 차이(overfitting gap)가 작을수록 견고한 전략.
train 3일 → test 2일 롤링
 		p-value
Bootstrap p-value
수익률을 500회 재샘플링하여 "우연히 0 이하가 나올 확률" 계산. 0.05 미만이면 통계적으로 유의미 ✓.
 		α
Benchmark Alpha (vs KOSPI)
KOSPI 지수 대비 초과수익률. 양수면 지수 대비 아웃퍼폼.
✓ Tier 1이 가장 신뢰 가능 (ground truth) · Nominal ≠ Tier 1 차이가 클수록 기존 시뮬이 과대/과소 평가 · p < 0.05 그리고 WF consistency 높은 전략 우선

Methodology

백테스트 조건

매매 시뮬
  • • 시초가 매수 → 일봉 high/low 기반 익절/손절 판정
  • • 익절: +5% / 손절: -3% / 미체결: 종가 청산
  • • 손절 우선 (보수적 시뮬)
  • • 초기 자본 1,000만원 균등 분할
데이터
  • • KRX OpenAPI (KOSPI + KOSDAQ 일별 OHLCV)
  • • Naver Finance (외국인/기관 수급)
  • • DART OpenAPI (공시)
  • • 일봉 기반 (분봉 historical 6일 한계)
메트릭
  • • Sharpe / Sortino / Calmar 비율
  • • 최대 드로다운 (MDD) + 회복 기간
  • • 승률 / Profit Factor / Win-Loss Ratio
  • • 최대 연속 승/패
한계 (정직)
  • • 일봉 기반 → high가 익절선에 닿으면 익절 판정 (실제론 못 닿을 수 있음)
  • • 거래비용/슬리피지 미반영
  • • 짧은 기간(1주~1개월)은 통계적 신뢰도 낮음
  • • 과적합 위험 — 실전 검증 별도 필요

Glossary

용어집 — 비전문가를 위한 안내

대시보드에 나오는 모든 메트릭/영어 용어/카테고리/기능에 대한 쉬운 설명입니다.

총 수익률

Total Return
백테스트 기간 동안 돈이 얼마나 늘었는지를 %로 표현합니다.
1,000만원 시작 → 1,352만원 = +35.2%
💡 가장 단순한 지표. 단, 짧은 기간일수록 운 요소 큼.

샤프 비율

Sharpe Ratio
"위험(변동성) 대비 수익이 얼마나 좋은가". 같은 수익률이라도 변동성이 작으면 Sharpe가 높아집니다.
공식: (수익률 - 무위험수익률) / 변동성 × √250
💡 1 이하 별로 / 1~2 괜찮음 / 2~3 좋음 / 3 이상 매우 우수. 단, 1주 같은 짧은 기간엔 30+도 나와서 과대평가됨

소르티노 비율

Sortino Ratio
Sharpe와 비슷하지만 "아래로 떨어지는 위험"만 나쁘다고 봅니다. 위로 튀는 변동성은 좋은 것이니 벌점 없음.
수익률 변동성 10% 중 하락은 3%만 → Sortino가 Sharpe보다 보통 큼
💡 Sharpe보다 현실적. 상승 변동성까지 위험으로 치는 Sharpe의 한계 보완.

칼마 비율

Calmar Ratio
연환산 수익률을 최대 낙폭(MDD)으로 나눈 값. "큰 손실 대비 몇 배 벌었나"
연 30% 수익 + MDD -15% → Calmar = 30/15 = 2.0
💡 1 이상 괜찮음, 3 이상 좋음. 고위험 고수익 전략일수록 낮아짐.

최대 낙폭 (MDD)

Max Drawdown
운용 중 가장 높았던 잔고에서 가장 낮았던 잔고까지 떨어진 최대 폭. 음수로 표시되며, 0에 가까울수록 좋음.
1,000 → 1,200 → 1,050 → 1,300 : MDD = -12.5% (1,200→1,050 구간)
💡 실전에서 가장 중요한 지표 중 하나. "이 낙폭을 버틸 수 있는가" 심리 테스트.

승률

Win Rate
전체 매매 중 수익으로 끝난 비율.
100번 중 65번 흑자 → 승률 65%
💡 50% 이상이면 괜찮지만 승률만 보면 안 됨. 9번 이기고 1번 큰 손실일 수 있음 → Profit Factor와 함께 봐야 함.

이익 요인

Profit Factor (PF)
총 번 돈을 총 잃은 돈으로 나눈 값. 1 초과면 흑자.
총이익 200만원 / 총손실 100만원 = PF 2.0 (이익이 손실의 2배)
💡 1.5 이상 괜찮음 / 2.0 이상 좋음. 승률은 낮아도 PF가 높으면 '크게 벌고 작게 잃는' 전략.

최대 연속 패

Max Consecutive Losses
연속으로 몇 번 지는 구간이 있었나. 심리적으로 견딜 수 있는 최대 연패 숫자.
5일 연속 손절 당하면 연속 패 = 5
💡 7 이상이면 탈락 기준. 실전에서 7번 연속 지면 "이 전략 틀렸나?" 의심 시작.

연환산 수익률 (CAGR)

Compound Annual Growth Rate
"이 수익률이 1년 내내 지속됐다면?" 을 복리로 환산한 값.
1주 +5% → 연 +1,200% (비현실적)
⚠️ 짧은 기간엔 무의미. 1년 이상 데이터에서만 신뢰 가능.

변동성 (연환산)

Volatility (Annualized)
수익률이 얼마나 널뛰는지를 측정. 일별 변동성에 √250(거래일) 을 곱해 연 단위로 환산.
변동성 20% = 연 평균 수익률 ± 20% 범위 내 변동
💡 작을수록 안정적. Sharpe 계산의 분모가 됨.

모멘텀

Momentum
"오르는 놈이 더 오른다"는 믿음. 상승 추세가 있는 종목을 추격하는 전략.
5일 이동평균 돌파 + 거래량 3배 = 매수

역추세

Contrarian
"너무 떨어진 것은 곧 반등한다"는 반대 방향 전략. 과매도 종목을 매수.
RSI 30 이하 대형주 매수

돌파

Breakout
주가가 저항선(저항 가격대)을 뚫고 올라가는 순간. 이때 관심과 매수세가 몰려 추가 상승 확률 ↑.
전고점 돌파 + 대량 거래 동반 = 매수

매수 / 공매도

Long / Short
Long은 "오를 거라 믿고 사는 것"(일반 매수), Short은 "내릴 거라 믿고 빌려서 파는 것"(공매도). 한국은 공매도 제약이 강함.
Long: 100만원에 사서 120만원에 판다
Short: 100만원에 빌려 팔고 80만원에 사서 갚는다

진입가 / 청산가

Entry Price / Exit Price
Entry는 처음 산 가격, Exit은 판 가격. 우리 시뮬에선 Entry는 장 시작 시초가.
시초가 10,000원 Entry → 10,500원 Exit (익절)

오버나이트

Overnight
장 마감 후부터 다음날 장 시작까지 포지션을 유지하는 것. 한국 시장은 오버나이트 수익률이 양수라는 학술 연구가 있음.
종가 매수 → 다음날 시초가 매도 = 오버나이트 수익

인트라데이

Intraday
장 중(09:00~15:30) 안에서 사고 파는 것. 단타의 본질.
시초가 매수 → 종가 청산 = 인트라데이 단타

슬리피지

Slippage
주문한 가격과 실제 체결가의 차이. 빠른 시장에서 주문이 밀리면서 발생.
10,000원에 사려 했는데 10,050원에 체결 = 슬리피지 +0.5%
⚠️ 현재 시뮬은 슬리피지 미반영 → 실전 수익률은 표시값보다 낮음

수급

Order Flow / Investor Flow
누가 사고 누가 파는지. 한국에선 외국인/기관/개인 3주체로 나눔. 외국인 순매수는 대표적인 "스마트 머니" 신호.
외국인 5일 연속 순매수 + 주가 상승 = 수급 강한 종목

DART 공시

DART Disclosure
금융감독원의 전자공시시스템. 상장사가 의무적으로 올리는 재무/계약/투자 발표. 호재 공시 후 주가 반응이 단타 기회.
대규모 수주 계약 공시 → 다음날 시초가 급등

익절 / 손절

Take Profit / Stop Loss
익절(익절선): 목표 수익 도달 시 매도. 손절(손절선): 허용 손실 도달 시 매도. 우리 시뮬은 +5% 익절, -3% 손절.
10,000원 Entry → 10,500원(+5%) = 익절, 9,700원(-3%) = 손절

백테스트

Backtest
과거 데이터로 "이 전략이 과거에 잘 먹혔는지" 검증하는 것. 과적합(overfitting) 주의 — 과거에만 잘 맞는 전략은 미래에 실패 가능.
2024-2026 KRX 데이터로 전략 돌려서 가상 수익률 계산
⚠️ 백테스트 ≠ 실전. 실전 검증(paper trading) 필수.

모멘텀

Momentum
상승 추세를 따라가는 전략. "이미 오르고 있는 종목"을 매수.
Alpha Momentum (MA5 돌파 + 거래량), KOSPI Intraday Momentum

역추세

Contrarian
많이 떨어진 종목의 반등을 기대. 과매도 구간 매수.
Beta Contrarian (RSI 역추세), BNF 낙폭과대, EOD Reversal Korean

돌파

Breakout
특정 가격/신고가를 돌파할 때 진입. 단타에서 가장 흔한 전략군.
Echo Frontier(시초가 갭), Volatility Breakout(변동폭), Turtle Breakout(N일 신고가)

이벤트

Event-Driven
공시/뉴스/실적 발표 같은 특정 이벤트에 반응하는 전략.
Gamma Disclosure (DART 공시), News Catalyst Timing

테마

Theme / Sector Rotation
당일 강세 섹터/테마를 찾아 그 안의 대장주 매수.
Delta Theme, Sector Rotation (KOSPI 업종 지수)

수급

Flow / Order Flow
외국인/기관 매매 동향 기반. "큰손들이 사는 걸 따라가자"
Foreign Flow Momentum (외국인 연속 순매수)

통계적 차익

Statistical Arbitrage
학술적으로 검증된 시장 비효율성 활용. 반복 가능한 통계적 이상현상.
Overnight ETF Reversal (한국 ETF overnight 양수 효과)

복합

Hybrid / Multi-Signal
여러 신호를 동시에 만족해야 매수. 단일 신호의 약점 보완.
Multi-Signal Hybrid (DART + 수급 + 거래량 AND 결합)

🏆 승급

PROMOTED
모든 승급 기준(6가지)을 통과한 전략. news-trading-bot 정식 통합 후보.
수익률 ≥5% · Sharpe ≥1.0 · 승률 ≥50% · MDD ≥-15% · 거래 ≥10 · PF ≥1.5 모두 통과
💡 통합 가이드 마크다운 자동 생성 → 수동 검토 후 실제 봇에 추가

⚠️ 관찰

WATCHLIST
일부 기준 미달이지만 완전히 탈락은 아님. 관찰하면서 데이터 쌓기.
수익률은 +8%지만 승률이 49%로 50% 미달

❌ 탈락

REJECTED
명확한 탈락 기준에 해당. 전략 자체를 재고하거나 폐기.
음수 수익률 / 7연속 패 / 30% 미만 승률 / 거래 없음 중 하나라도 해당

⏳ 대기

PENDING
데이터가 부족해서 평가 자체가 불가능한 상태.
거래 수 5 미만 또는 거래일 5일 미만

참신성 점수

Novelty Score (0~10)
전략이 얼마나 새로운 아이디어인가. 기존 6개 전략과 얼마나 다른지에 대한 내부 평가.
Multi-Signal Hybrid(3중 결합) = ★9, Turtle Breakout(클래식 축소) = ★6

신뢰도 등급

Trust Level
전략 출처의 신뢰성을 5단계로 평가. 학술 논문이 가장 높고 익명 블로그가 가장 낮음.
★★★★★ 동료심사 논문 / ★★★ 일반 블로그 / ★ 익명 포스트

시초가 매수

Market Open Entry
장이 시작하는 09:00의 첫 체결가격으로 매수한다고 가정.
전날 장 마감 후 전략이 종목 선정 → 다음날 09:00 시가에 자동 매수

익절 +5%

Take Profit at +5%
매수 후 가격이 +5%에 도달하면 즉시 매도. 우리 시뮬은 일봉 고가가 이 가격에 닿으면 익절로 판정.
10,000원 매수 → 10,500원 도달 → 익절 매도

손절 -3%

Stop Loss at -3%
매수 후 가격이 -3%로 하락하면 즉시 매도. 일봉 저가가 이 가격에 닿으면 손절로 판정.
10,000원 매수 → 9,700원 도달 → 손절 매도

종가 청산

Market Close Exit
익절/손절 모두 도달 안 하면 장 마감(15:30) 종가에 자동 매도. 당일 청산이 원칙.
10,000원 → 고가 10,300원, 저가 9,900원 → 10,200원 종가에 매도

손절 우선 규칙

Stop-Loss-First Rule
일봉만으로는 장중 가격 경로를 모름. 보수적으로 "저가가 먼저 닿았다"고 가정 → 손절 우선.
고가가 익절선 초과 + 저가가 손절선 도달 = 손절로 판정 (수익 기회 버림)
💡 실제 결과가 시뮬보다 나쁘지 않게 하는 안전장치

일봉 기반 시뮬

Daily Bar Simulation
분봉이 아닌 일봉(하루 단위 시/고/저/종)으로 시뮬. 데이터 한계상 장기 백테스트에 필수.
분봉 = 1분 단위 가격 추적 (정확하지만 6일 한계) / 일봉 = 하루 한 줄 요약 (무제한)
⚠️ 정확도 ~80%. 익절 과대평가 위험 → Phase 8에서 분봉 정밀 시뮬 예정

초기 자본 1,000만원

Initial Capital
모든 전략이 같은 조건에서 경쟁할 수 있도록 초기 자본을 통일. 선정 종목 수로 균등 분할.
5종목 선정 → 종목당 200만원 배분

거래비용/슬리피지 미반영

No Costs / No Slippage
수수료(~0.015%)와 슬리피지(~0.1%)는 현재 시뮬에 반영 안 됨. 실전 수익률은 표시값보다 약간 낮음.
시뮬 +5% → 실전 약 +4.5% 수준
⚠️ 거래 빈도가 높은 전략일수록 영향 큼
