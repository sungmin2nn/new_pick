# Quality Checklist - 점수 기반 품질 검증

> quality_gate.md와 연동되는 세부 체크리스트

---

## 1. 검증 프로세스 (점수 기반)

```
에이전트 작업 완료
       ↓
   결과물 수신
       ↓
┌─────────────────────────────────┐
│  영역별 점수 산출               │
│  (evaluate_agent 또는 직접)     │
└─────────────────────────────────┘
       ↓
   과락 체크
       ↓
┌─ 과락 있음 (1개 이상)
│      ↓
│  🛑 즉시 불통
│      ↓
│  과락 영역 + 감점 사유 명시
│      ↓
│  동일 에이전트 재작업 요청
│
└─ 과락 없음
       ↓
   총점 계산
       ↓
   ┌─ 총점 < 커트라인 → ⚠️ 불통 + 재작업
   │
   └─ 총점 ≥ 커트라인 → ✅ 통과 + 다음 단계
```

## 4. 재전달 프로토콜

### 4.1 재시도 정책 (점수 기반)

| 시도 | 조치 | 추가 지원 |
|------|------|-----------|
| 1차 불통 | 감점 내역 + 구체적 수정 지시 | - |
| 2차 불통 | 다른 접근 방식 제안 + 관련 이슈 참조 | 코드 예시 제공 |
| 3차 불통 | 사용자에게 보고 + 수동 개입 요청 | 전체 점수 이력 첨부 |

### 4.2 최종 실패 보고

```
═══════════════════════════════════════════════════════
       품질 검증 최종 실패 - 수동 개입 필요
═══════════════════════════════════════════════════════

작업: {작업 설명}
에이전트: {에이전트명}
시도 횟수: 3/3

점수 이력:
  1차: 72점 (코드품질 8점, UX 12점)
  2차: 76점 (코드품질 11점, UX 13점)
  3차: 78점 (코드품질 12점, UX 14점)

지속 미달 영역:
- 총점 78점 < 커트라인 80점

- 1차: 타입 오류 3개 → 1개로 감소
- 2차: Empty State 추가, 타입 오류 해결

남은 문제:
- 성능 영역 7점으로 개선 여지 없음
- 문서화 4점 유지

권장 조치:
- 성능 최적화 전문가 검토 필요
- 또는 Phase 1 기준(70점)으로 임시 통과 고려
```
