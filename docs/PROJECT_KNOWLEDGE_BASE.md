## 1. 프로젝트 개요

### 1.1 목표

실매매 전 **전략 검증**과 **데이터 축적**을 위한 자동화 시스템

```
전략 수립 → 종목 선정 → 성과 측정 → 데이터 분석 → 전략 개선
    ↑                                               ↓
    └───────────────── 반복 ─────────────────────────┘
```

### 1.2 로드맵

| Phase | 단계 | 상태 | 목표 |
|-------|------|------|------|
| 1 | 전략 수립 및 백테스팅 | **진행 중** | 실데이터 기반 전략 검증 |
| 2 | 모의투자 (페이퍼 트레이딩) | 대기 | HTS 모의투자로 실전 시뮬레이션 |
| 3 | 소액 실매매 | 대기 | 10~50만원 소액 테스트 |
| 4 | 본격 자동매매 | 대기 | 한국투자증권 API 연동 |

### 1.3 핵심 원칙

```
1. 데이터 기반 의사결정
   - 감이 아닌 실제 성과로 판단
   - 최소 1개월 데이터로 검증

2. 점진적 접근
   - 백테스팅 → 모의투자 → 소액 → 본격 매매
   - 급하게 실매매 진입하지 않음

3. 문서화 필수
   - 모든 결정에 사유 기록
   - 나중에 왜 이렇게 했는지 추적 가능
```

```
    - 종가 청산 15:20

📉 BNF 낙폭과대 (bnf_dashboard.html)
    - 낙폭과대 역추세 전략
    - 분할 매수 3회 / 분할 매도 3회
    - 트레일링 스탑

📚 가이드 (system_guide.html)
    - 시스템 사용법
    - 점수 시스템 설명
    - 매매 전략 가이드
```

### 2.3 데이터 흐름

```

    │
    ├─ stock_screener.py 실행
    ├─ 145점 점수 계산
    ├─ 상위 5개 종목 선정
    └─ morning_candidates.json 저장
        │
        ↓
[09:00~15:30] 장중 모니터링
    │
    ├─ 시초가 매수 가정
    ├─ 익절/손절 도달 감시
    └─ 종가 청산 확인
        │
        ↓
[16:30] 결과 수집
    │
    ├─ intraday_collector.py 실행
    ├─ 시초가, 고가, 저가, 종가 수집
    ├─ 익절/손절 분석
    └─ data/intraday/intraday_YYYYMMDD.json 저장
        │
        ↓
[대시보드] 시각화
    │
    ├─ 누적 수익률 표시
    ├─ 승률, MDD, 손익비 계산
    └─ 최근 거래 내역 표시
```

## 4. 투자 전략

### 4.1 페이퍼 트레이딩 전략

#### 전략 1: 모멘텀 (Momentum)
```
조건: 상승세 + 뉴스/공시 + 높은 점수
진입: 시초가 매수
청산: 익절 +5% / 손절 -3% / 종가 15:20
```

#### 전략 2: 대형주 역추세 (Large-Cap Contrarian)
```
조건: 전일 -1% 이상 하락 + 5만원 이상 대형주
원리: 대형주는 일시적 하락 후 평균 회귀
진입: 시초가 매수
청산: 익절 +5% / 손절 -3%
```

#### 전략 3: DART 공시
```
조건: 호재 공시 발표 (실적, 계약, 투자)
점수: 공시 40점 반영
진입: 시초가 매수
```

#### 전략 4: 테마/정책
```
조건: AI, 반도체, 2차전지 등 테마 매칭
점수: 테마 15점 반영
진입: 시초가 매수
```

### 4.2 BNF 낙폭과대 전략

```
[진입 조건]
- 전일 대비 -5% 이상 급락
- 거래대금 500억 이상
- 뉴스/공시 확인 (악재 아닌지)

[분할 매수]
1차: -5% 하락 시 30%
2차: -7% 하락 시 40%
3차: -10% 하락 시 30%

[청산 규칙]
- 트레일링 스탑: 고점 대비 -2%
- 분할 매도: +3%, +5%, +7%에서 각각 1/3
```

### 4.3 매매 규칙 (공통)

| 구분 | 규칙 | 비고 |
|------|------|------|
| 익절 | +5% | 목표 수익률 |
| 손절 | -3% | R:R = 1.67:1 |
| 종가 청산 | 15:20 | 미도달 시 |
| 최대 종목 | 5개 | 분산 투자 |
| 종목당 투자 | 20만원 | 초기 설정 |
