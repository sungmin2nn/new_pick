## 1. 프로젝트 개요

### 1.1 프로젝트 목표

| 단계 | 목표 | 상태 |
|------|------|------|
| Phase 1 | 종목 선정 로직 개발 및 백테스트 검증 | ✅ 완료 |
| Phase 2 | 페이퍼 트레이딩 (가상 매매 기록) | 🔄 진행 예정 |
| Phase 3 | 데이터 수집 및 전략 검증 (1~12개월) | ⏳ 대기 |
| Phase 4 | 실제 자동매매 구현 | ⏳ 대기 |

### 1.2 핵심 원칙

```
1. 데이터 기반 의사결정
   - 감이 아닌 백테스트 결과로 판단
   - 최소 1~3년 데이터로 검증

2. 점진적 접근
   - 페이퍼 트레이딩 → 소액 실매매 → 본격 매매
   - 급하게 실매매 진입하지 않음

3. 다중 전략 비교
   - 단일 전략 맹신 금지
   - 3~5개 전략 동시 추적 후 선택

4. 문서화 필수
   - 모든 결정에 사유 기록
   - 나중에 왜 이렇게 했는지 추적 가능하게
```

### 1.3 제약 조건

| 항목 | 제약 | 사유 |
|------|------|------|
| 초기 자본 | 100만원 | 소액으로 시작하여 리스크 관리 |

## 3. 중요 의사결정 기록

### 3.1 전략 선택: 대형주 역추세

**결정일**: 2024-12-28
**결정 내용**: 10개 전략 중 "대형주 역추세" 전략 채택

**배경**:
```
기존 전략 문제점:
- 대형주 + 뉴스/공시 기반
- 1년 수익률: -34.2%
- 승률: 17.4%
```

**비교 결과**:
| 전략 | 수익률 | 승률 | 선택 |
|------|--------|------|------|
| 대형주 매수 (기존) | -81.5% | 35.6% | ❌ |
| 중소형 역추세 | +23.7% | 47.1% | - |
| 모멘텀 | -27.0% | 36.0% | ❌ |
| **대형주 역추세** | **+166.4%** | **43.9%** | ✅ |

**선택 사유**:
1. 3년 연속 양수 수익률 (2022~2024)
2. 위기 상황에서도 상대적 안정성
3. 대형주라 유동성 문제 없음
4. 단순하고 실행 가능한 규칙

---

### 3.2 파라미터 선택: +3% / -1.5%

**결정일**: 2024-12-28
**결정 내용**: 익절 +3%, 손절 -1.5%

**테스트 결과**:
| 익절 | 손절 | 수익률 | 승률 |
|------|------|--------|------|
| 3% | 2% | +452% | 43.4% |
| **3%** | **1.5%** | **+757%** | **47.8%** |
| 5% | 3% | +166% | 43.9% |
| 5% | 2% | +321% | 42.1% |

**선택 사유**:
1. R:R 비율 2:1 (익절이 손절의 2배)
2. 가장 높은 연간 수익률
3. 손절 빠르게 → 손실 최소화

---

### 3.3 페이퍼 트레이딩 우선 결정

**결정일**: 2024-12-28
**결정 내용**: 실매매 전 1~3개월 페이퍼 트레이딩 수행

**사용자 요구사항**:
```
"현재는 전략을 구현하고 장 종료 후에 다음 날 매매 종목 선정과
당일 매매 종목에 대한 액션 결과 정리만 하고,
1달~1년 정도 자료 수집 후 실자동화 매매 구현 예정"
```

**결정 사유**:
1. 백테스트 ≠ 실제 매매 (슬리피지, 체결 실패 등)
2. 실제 데이터로 전략 재검증 필요
3. 리스크 없이 전략 개선 기회 확보

## 4. 용어 및 규칙 정의

### 4.1 전략 용어

| 용어 | 정의 | 예시 |
|------|------|------|
| **역추세 (Contrarian)** | 전일 하락한 종목을 매수하여 반등을 노리는 전략 | 전일 -2% 하락 종목 매수 |
| **모멘텀 (Momentum)** | 상승 추세 종목을 따라 매수하는 전략 | 전일 +5% 상승 종목 매수 |
| **대형주** | 시가총액 10조원 이상 또는 주가 5만원 이상 | 삼성전자, SK하이닉스 |
| **중소형주** | 시가총액 1조원 미만 또는 주가 5만원 미만 | - |
| **익절 (Profit Taking)** | 목표 수익률 도달 시 매도 | +3% 도달 시 청산 |
| **손절 (Stop Loss)** | 손실 한도 도달 시 매도 | -1.5% 도달 시 청산 |
| **R:R 비율** | 익절 대 손절 비율 (Risk:Reward) | 3:1.5 = 2:1 |

### 4.2 매매 규칙

```
[대형주 역추세 전략 - 최종 규칙]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

■ 종목 선정 조건
  1. 가격: 50,000원 이상
  2. 전일 등락률: -1% 이상 하락
  3. 거래대금: 500억원 이상

■ 매수 규칙
  - 시점: 당일 시가 (09:00~09:05)
  - 수량: 자본금 / 종목수 / 시가
  - 최대 종목: 5개

■ 매도 규칙
  - 익절: 매수가 대비 +3% 도달 시 즉시
  - 손절: 매수가 대비 -1.5% 도달 시 즉시
  - 시간 청산: 14:30까지 미체결 시 종가 청산

■ 예외 규칙
  - 코스피 -3% 이상 하락일: 매매 중단
  - 부정 뉴스 2건 이상: 해당 종목 제외
```

### 4.3 점수 체계

```
[역추세 전략 점수 배점 - 총 100점]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

   -1% ~ -3%: 10~20점
   -3% ~ -5%: 20~35점
   -5% 이상: 35점 (만점)

2. 거래대금 점수 (25점)
   500억 이상: 5점
   1000억 이상: 15점
   3000억 이상: 25점

3. 시가총액 점수 (15점)
   1조 이상: 5점

   10조 이상: 15점

4. 거래량 변화 점수 (15점)

   평균 대비 2배: 15점

5. 가격대 적합성 (10점)
```
