### 6.2 stock_screener.py 수정안
```python
def calculate_reversal_score(row):
    """역추세 전략 점수 계산"""
    price = row['종가']
    change = row['등락률']

    # 가격대 필터
    if not (3000 <= price <= 50000):
        return 0

    # 하락폭 필터
    if change > -2:
        return 0

    # 점수: 하락폭이 클수록 높은 점수
    score = abs(change) * 10
    return score
```
